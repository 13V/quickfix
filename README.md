# QuickFix Plumbing - Ultra-Premium Website

A professional, trust-building website for QuickFix Plumbing featuring 24/7 emergency services, modern design, and conversion-focused features.

## Features

- **Professional Blue Theme**: Trust-building color scheme with blue primary and orange emergency accents
- **24/7 Emergency Focus**: Prominent emergency service badges and pulsing call-to-action buttons
- **Trust Indicators**: Licensed, insured, 5-star ratings displayed throughout
- **Mobile-First Design**: Fully responsive with floating emergency button
- **Service Showcase**: Comprehensive plumbing services with detailed descriptions
- **Quote Request System**: Easy-to-use form with urgency levels
- **Service Areas**: Clear coverage display for Adelaide suburbs
- **Customer Testimonials**: Social proof with 5-star reviews

## Local Development

Simply open `index.html` in a modern web browser to view the website.

For a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server
```

Then navigate to `http://localhost:8000`

## Deployment

This is a static website and can be deployed to:
- **Vercel**: Connect your GitHub repository
- **Netlify**: Drag and drop the folder
- **GitHub Pages**: Enable in repository settings
- **Any web hosting**: Upload files via FTP

## Customization

### Contact Information
Update the following throughout the site:
- Phone number: `0400 000 000`
- Email: `info@quickfixplumbing.com.au`
- Business address and ABN

### Service Areas
Edit the service areas section in `index.html` to match actual coverage.

### Images
Replace the placeholder images:
- `hero.png` - Professional plumber image
- `emergency.png` - Emergency service photo
- `hot-water.png` - Hot water system installation

### Colors
Edit CSS custom properties in `styles.css`:
```css
:root {
    --primary: #1E40AF;
    --accent: #F97316;
}
```
